Screenshot of the homepage.

My first webpage was created for the world wide web design and managment course. It is an introductory course that focuses on teaching the fundamentals of HyperText Markup Language and Cascading Style Sheets. Essentially all the assignments built on eachother over the semeseter and the end result is our very first website.

Screenshot of the personal interest page

The website is very barebones and lacks styling, but I was able to learn alot about how to start building websites from scratch. The purpose of the course was not to create something extravagant, but to understand the fundamentals of web development. Topics such as tags, structure, and deployment.

Screenshot of FileZilla.

In this course I also got experience with a file transfer program. We used FileZilla and it was a refresher for me. I have FileZilla in the past when I helped administrate gaming servers. The program is easy to use and I think it is a good for individuals who want to start learning how to create websites.
